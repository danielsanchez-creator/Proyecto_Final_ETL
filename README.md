Título: Pipeline ETL Multimodal - Análisis de Churn Olist
Descripción:
Desarrollo de un ecosistema de datos profesional para el procesamiento de 113,314 registros del marketplace brasileño Olist. El proyecto integra variables logísticas y análisis de sentimiento (NLP) para la preparación de modelos predictivos de deserción.

Características Principales:

Arquitectura Modular: Separación de fases (Extract, Transform, Load,pipeline,visualized,audit).

Auditoría de Datos: Implementación de un módulo de KPIs de Ingeniería (Tasa de vinculación e integridad semántica).

Enriquecimiento: Conversión monetaria a USD, normalización de texto con unidecode y categorización de sentimiento.

Cómo ejecutar:

Instalar dependencias: pip install -r requirements.txt

Ejecutar el pipeline: python main.py
